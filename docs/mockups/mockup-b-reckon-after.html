What actually changed. Content identical to Mockup A — every string, every data point. Only the styling is different.
  • Primary buttons become StadiumBorder pills (12 px rounded rect → fully round).
  • Display type switches to Lora weight 700 with tightened letter-spacing — assertive headings instead of soft ones.
  • Section headers drop from hearth to linen-500 labelSm, an Oh convention (hearth is reserved for interactive).
  • Six palette values align to spec (hearth-100 +→ #F8E8E3, linen-100 +→ #F5EFE6, linen-300 +→ #D4C5B0, linen-900 +→ #2C1810).
  • Cards land at OhRadii.lg (12 px, from 16 px) — tighter, less "chat app".
  • New sliderTheme being contributed back to OhTheme: hearth track, hearth-600 thumb, 15% hearth overlay. Rendered in the reveal card.

Same app. Same content. Same accent. What changes is grammar — Reckon stops being a warm-toned app and starts being an OpenHearth app. The delta is quiet because it was designed to be; that's the point of a design system.
