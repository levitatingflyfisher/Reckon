Lullaby is a baby-tracker — different app, different audience, totally different content. The whole design system adoption is one constructor argument: appAccent: sage500. Every card, every type ramp, every radius, every input style is inherited.

Same grammar, different voice. Lora displays, Nunito UI, linen surfaces, 12 px card radii, pill buttons — identical. The only deltas are the accent (sage 500 instead of hearth 500) and the app-specific content. A parent who uses both Reckon and Lullaby feels instant familiarity without the apps feeling like they're the same app.
