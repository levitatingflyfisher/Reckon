Reckon & the OpenHearth design system — three mockups.

The existing ohStyle/openhearth_design package is the source of truth. Reckon is the first consumer. These three pages walk the migration: what Reckon looks like now, what it looks like after the package lands, and what the next app looks like once the grammar is shared.

Mockup A · Baseline

Reckon before.

Three screens — intake, record, reveal — rendered with Reckon's current lib/core/theme. Hand-picked hearth palette that drifts from spec in six values. 12 px rounded rectangles. Lora weight 500.

hearth accent custom AppTheme 230 LOC
Mockup B · Migration

Reckon after.

Same content, OhTheme grammar. Every palette value aligned. StadiumBorder pills. Lora weight 700 for displays. New sliderTheme contributed back to openhearth_design. lib/core/theme deleted.

hearth accent OhTheme.light() −230 LOC
Mockup C · Propagation

Lullaby after.

A different app — baby tracker, different audience — adopts the system with one constructor argument: appAccent: sage500. Every card, type ramp, radius, and input style comes for free. Personality lives in accent and content, not primitives.

sage accent OhTheme.light(appAccent:…) ~90 LOC
